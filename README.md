# Tamamon — Releases

Downloads for **Tamamon**, a macOS desktop pet that grows as you code with Claude.

- **Website:** https://tamamons.com
- **Latest build:** see the **[Releases](../../releases/latest)** tab → `Tamamon.dmg`

## Install
1. Download `Tamamon.dmg` from the latest release.
2. Open it and drag **Tamamon** into Applications.
3. First launch: if macOS warns about an unidentified developer, right-click → **Open** once.

## Requirements
macOS 12 (Monterey)+ · Apple Silicon. Signed & notarized (Developer ID).

This repository hosts **release binaries only** — the app source is private.
